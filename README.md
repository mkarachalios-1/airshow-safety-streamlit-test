# Airshow Safety & Excellence Database (Streamlit)

Interactive, 5M-aligned repository of airshow accidents/incidents (1908–2025).

## Deploy on Streamlit Community Cloud
1. Create a **public GitHub repo** and upload the contents of this folder (`app.py`, `requirements.txt`, `data/`).
2. Go to https://share.streamlit.io → **New app** → pick your repo/branch → `app.py` → Deploy.

## Embed in EAC members area
Place this iframe in your members-only page (replace with your deployed app URL):

```html
<iframe src="https://YOUR-APP.streamlit.app/?embed=1" width="100%" height="1200" style="border:0;" allow="clipboard-read; clipboard-write; fullscreen"></iframe>
```
